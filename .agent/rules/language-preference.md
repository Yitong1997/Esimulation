# 中文语言偏好

用户要求所有的回复、文档、代码注释、以及工具输出的解释说明都必须使用**简体中文**。

## 规则
1.  **代码注释**: 必须使用中文。
2.  **文档 (.md)**: 必须使用中文撰写。
3.  **对话回复**: 必须使用中文。
4.  **例外**: 
    - 代码中的变量名、函数名、类名等保留英文。
    - 专有名词（如 PROPER, Optiland, Zemax）保留英文。
    - 文件路径保留原文。
