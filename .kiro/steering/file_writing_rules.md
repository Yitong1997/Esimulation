# 文件写入规则

## 长文件写入规范
- 当文件内容超过 20 行时，必须使用多次写入方式
- 首先使用 fsWrite 创建文件并写入前 20-50 行
- 然后使用 fsAppend 分批追加剩余内容
- 每次追加不超过 50 行

inclusion: always
